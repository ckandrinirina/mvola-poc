# Project Skills

Type `/help` in Claude Code for interactive help, or read below.

## Commands

| Command | What it does | Input | Output |
|---------|-------------|-------|--------|
| `/design` | Refine spec into architecture docs | spec file | `docs/architecture/` |
| `/team` | Generate expert + guide skills | `docs/architecture/` | `experts/`, `guides/` |
| `/plan` | Create epics, stories, roadmap | spec file | `tasks/YYYY-MM-DD_*/` |
| `/publish` | Push to GitHub Issues | tasks folder | GitHub Issues |
| `/track` | Show progress dashboard | — | status, next, progress |
| `/build` | Implement a story (TDD) | story file | source code + tests |
| `/parallel-build` | Implement multiple ready stories in parallel | — or story IDs | parallel worktrees + QA + conflict report |
| `/fix` | Fix a bug in a story | story file | minimal fix + regression test |
| `/ship` | Commit, PR, update issues | story file | commit + PR + issue updates |

## Workflow

```
/design → /team → /plan → /publish → /track → /build → /ship
                                                  ↓
                                                /fix → /ship
```

```
├── publish/SKILL.md      # Epics → GitHub Issues
├── track/SKILL.md        # Progress dashboard
├── build/SKILL.md        # TDD story implementation
├── fix/SKILL.md          # Bug fix workflow
├── ship/SKILL.md         # Commit, PR, issue updates

│   ├── backend/          #   /expert-backend

    ├── rust/             #   Auto-loaded for .rs files
    ├── cpp/              #   Auto-loaded for .cpp/.h files
    └── .../              #   One per detected technology
```

## Quick Start

```bash
# First time setup
/design docs/specifications.md
/team
/plan docs/specifications.md
/track next
/build

# Adding a feature later
/design docs/new-feature.md
/team --regenerate
/plan docs/new-feature.md
/build

# Fixing a bug
/fix tasks/.../stories/01_setup.md
/fix                                  # interactive story picker

# Shipping (commit + PR + issue updates)
/ship tasks/.../stories/01_setup.md   # with story context
/ship                                 # standalone commit
```
